PICO 8

Type de projet : jeu
Langage : LUA
Taille de l'équipe : 3 personnes
Durée : 2 semaines

LIEN REPOSITORY GITHUB VERSION LUA
LIEN REPOSITORY GITHUB VERSION NAVIGATEUR
Pico-8 est une console imaginaire qui permet de créer, partager, et jouer à des jeux miniatures qui évoquent avec nostalgie les débuts du jeu vidéo grand public.
Le but de ce jeu réalisé en équipe est de récolter les cadeaux et les ammener au Père Noël.

Compétences sollicitées :
Gestion des ressources graphiques et sonores
Gestion des collisions
Gestion des inputs utilisateur
Suivi du joueur avec la caméra
Animation personnage et dialogues
Menu avec écran-tire
