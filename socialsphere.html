Social sphere



Type de projet : réseau social
Langage : PHP, Mysql, HTML & CSS
Taille de l'équipe : 3 personnes
Durée : 2 semaines

LIEN REPOSITORY GITHUB
Social sphere est un réseau social où les utilisateurs peuvent créer un compte, s'y connecter et réaliser des publications. Les utilisateurs ont aussi la possibilité de mettre un like sur une publication et s'abonner à un autre utilisateur.

Compétences sollicitées :
Gestion des authentifications, des sessions et des inscriptions
Utilisation d'une base de données SQL et MAMP
Animation du bouton like
Choix d'avatar pour l'utilisateur
