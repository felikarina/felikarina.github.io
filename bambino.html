Bambino



Type de projet : application web
Langage : Javascript & MongoDB
Taille de l'équipe : 1 personne
Durée : en cours


Bambino est une application web permettant aux parents qui confient leurs enfants à une crèche d'avoir des informations de la part du personnel et de communiquer. Le personnel est amené à rentrer des informations sur le sommeil, l'alimentation et les incidents.

Compétences sollicitées :
- gestion compte utilisateur
- mobile responsive pour une utilisation sur tablette et téléphone
- navigation par onglet
- création et gestion d'une base de données non SQL
- rédaction d'un cahier des charges
