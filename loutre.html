Adopte ta loutre



Type de projet : application mobile
Langage : Kotlin, Java(Spring)
Taille de l'équipe : 4 personnes
Durée : 6 semaines

LIEN REPOSITORY GITHUB
Adopte ta loure est une application mobile android proposant à l'utilisateur de parrainer une loutre dans un zoo.

Compétences sollicitées :
- Gestion du Main Activity et des fragments
- Animation du texte
- Conception d'une base de données et des routes
- Utilisation du Recycler View pour la liste
