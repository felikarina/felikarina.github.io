GoRest!



Type de projet : extension
Langage : Javascript, HTML & CSS
Taille de l'équipe : 3 personnes
Durée : 2 semaines

LIEN REPOSITORY GITHUB
GoRest est une extension pour le nagivateur Chrome invitant l'utilisateur à faire une pause au bout d'un temps modulable en floutant le contenu de la page web et en faisant apparaître une fenêtre possédant un bouton qui ferme entièrement le navigateur.

Compétences sollicitées :
Gestion et affichage d'un minuteur
Déclenchement d'une fenêtre pop-up
Paramétrage du temps par l'utilisateur
Injection de scripts dans le DOM des pages web
Utilisation du Manifest V3
